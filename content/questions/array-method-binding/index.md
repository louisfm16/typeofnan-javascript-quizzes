What gets logged in the following scenario?

```javascript
const map = ['a', 'b', 'c'].map.bind([1, 2, 3]);
map(el => console.log(el));
```

<!-- explanation -->

`Object.bind` will bind the `this` keyword to the provided value (in this case, that's `[1, 2, 3]`).
